## 練習 れんしゅう

```ts
1.古屋先生は今日はいらっしゃいません。　// いる
⇒ 古屋老师今天不在。
2.先生は明日の授業が8時からとおっしゃいました。// 言う
⇒老师说明天的课8点开始。
3.どうぞたくさん召し上がってください。// 食べる・飲む
请吃（请开动）
4.日本でどんな研究をなさいましたか。// する
在日本做什么研究。
5.今日の新聞をご覧になりましたか。// 見る・読む
您看过今天的报纸了吗?
6.これは山田先生がお書きになった日本語文法の本です。　// 書く
这是山田老师写的日语语法书。
7.李さんは佐藤さんの結婚式にご出席になりますか。// 出席する
小李出席佐藤的婚礼吗。
8.当店ではクレジットカードはご利用になれません。// 利用する
本店不能使用信用卡。
```

## 〜場合は〈假设〉

意义:用于举出可能出现或遇到的情况。后项一般为在此情况下的判断、对策或办法。  
译文 :当 ……时; ……的情况下  
接续: 动词、形容词的连体形+ 場合は名词+ の+ 場合は

```ts
（1）遠足が丸1日の場合は土曜日、半日の場合は日曜日がいいんですが……。
郊游一整天的话周六比较好，半天的话周日比较好。
（2）王さんの都合が悪い場合は、ほかの日にしましょう。
小王不方便的话，改天吧。
（3）電話が通じない場合はメールで連絡してください。
如果电话不通，请通过电子邮件与我们联系。
（4）雨が降った場合は遠足を中止します。
如果下雨，取消远足。
```

> 「〜場合は」不用于表达已经发生的事情。

```ts
（5） 図書館にいた｛X 場合は/ X 場合に/〇ときに｝、地震が起きた。
```

## って〈引用〉

意义: 表示引用 ,相当于「〜と（言う/思う）」。  
接续: 句子+ って

```ts
（1）高橋さんと渡辺さんも行きたいって言ってたんだ。
 高桥和渡边也说想去。
（2）李さんは、すぐ来るって言ってますよ。
小李说马上就来。
（3）つまらないかなって思ってたんだけど、おもしろかったよ。
我还以为会无聊，结果很有意思。
```

> ★ 口语中,表示言说的动词也可以省去,只保留「って」。

```ts
（4）李さんは傘を電車に忘れたって。
 小李把伞忘在电车上了。
```

⇒王さんは日本語がとても簡単だって思っている。
2.我们约好了一起加油。
⇒私たちは一緒に頑張るって約束した。

## 〜と言っていた

意义: 用于说话人向对方转述他人的话语。  
译文 :说了……; 说过……  
接续: 简体句子+ と言っていた

```ts
（1）高橋さんと渡辺さんも来たいって言ってたんだ……。
高桥和渡边也说想来。
（2）部長:田中さん、いないのか。
秘書 :ええ、今日から上海へ出張すると言っていました。
 部长：田中不在吗。
 秘书：是的，他说今天要去上海出差。
（3）高橋：李さんはまだですか。
王 : うん、ちょっと遅れてくるって言ってましたよ。
高桥: 小李还没回来吗。
王: 嗯，他说会晚一点到。
（4）弟 : えっ 、お姉ちゃん行かないの? 一緒に行くって言ってたのに。
高 橋 : ごめん。
弟弟: 诶，姐姐不去吗?你说过你会和我一起去的。高桥: 对不起。
```

> 第三人称也可以用「と言いました」

```ts
（5）林さんはこのテニスが難しいと言いました。
（6）お父さんは今夜残業すると言いました。
```

> 注 意 「〜と言っていた」与 「〜と言った」的区别:

```ts
a. 一緒に行くって言っていた。 （侧重点在于传达、转述的内容）
b. 一緒に行くって言った。 （侧重点在于“说了”这一动作）
```

1.你明明说过不会放开我的手的。「はなす」
⇒私の手を離さないと言っていたのに。
2.山田说过喜欢那个公园。
⇒山田さんはあの公園が好きだと言っていた。

## V たらどうですか

意义:向对方提出建议或征求意见。  
译文 : ……怎么样; 你看……如何  
接续: V た+ たらどうですか

```ts
(1) せっかくだから、日本人の語学留学生も誘ったらどうですか。
难得邀请日本的语言留学生怎嚒样。
(2) その仕事は、王さんに頼んだらどうですか。
那份工作拜托小王做怎嚒样。
(3) 眠いんですか。コーヒーを飲んだらどうですか。
 困吗。喝杯咖啡怎么样。
(4) さっきからずっと勉強していますね。少し休んだらどうですか。
  从刚才开始一直在学习。稍微休息一下怎么样。
```

## 練習 れんしゅう

```ts
1.有时间的话做下作业怎么样。
⇒時間があったら、宿題をしたらどうですか。
2.最近身体有点疲倦，多吃点肉怎么样。【だるい】
⇒最近体がだるいので、もっと肉を食べたらどうですか。
```

## N から

意义:表示动作发出的主体或者顺序的起点。  
译文 : 由……来 ; ……先; 由……开始

```ts
(1) じゃあ、私からご都合を伺いましょうか。
那么，我来拜访您方便吗。
(2) 私から先生に報告します。
 由我向老师报告。
(3) 準備ができた人から入ってください。
 请准备好的人进来。
(4) 結論から先に言うと、こうなります。
先从结论说起，就是这样。
```

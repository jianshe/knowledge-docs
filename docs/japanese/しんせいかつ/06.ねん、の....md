> 疑問詞： 何年 なんねん いまはなんねんですか。  
> 10 年、20 年······90 年均读成-4 型 // -4 "负数式"中的数字表示从后往前数时高读的最后一个音节的位置。

```ts
（1）あいうえお0＝あいうえお0
（2）あいうえお4＝あいうえお-2
（3）あいうえお5＝あいうえお-1
```

## の＜同位＞

意义： 「の」用于两个名词之间，表同位关系，即「の」前后的名词所指相同。

接续： 身份性质+の+专有名词； 性质，关系名词

```ts
（1）この方は留学生の高橋さんです。
（2）こちらはわたしの友達の王さんです。 // 友達（ともだち）
（3）こちらは高橋さんの恋人（こいびと） の 王さんです。
```
